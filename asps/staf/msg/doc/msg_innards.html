MSG_Check
Lookup a prefix, define new ones, return fast-ref ID.
MSG_Enter
Define new prefixes and assign fast-ref IDs.
MSG_Enter_Class
Define new class and assign an ID.
MSG_Find
Lookup a prefix in the MSG prefix index, return fast-ref ID.
MSG_Find_Class
Lookup a class in the MSG class index, return ID.
MSG_Get_Class
Strip the class off of a prefix and return it.
MSG_Get_Prefix
Strip the prefix off a message and return it.
MSG_Incr
Increment a prefix's counter, referenced by ID.
MSG_Parse_Prefix
Parse a prefix into a string and a number.
