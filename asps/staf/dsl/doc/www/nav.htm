Functions for navigation of datasets

dsDatasetEntry - return a handle for a dataset entry

dsDatasetEntryCount - return the number of entries in a dataset

dsDatasetMaxEntryCount - return the maximum number of entries for a dataset

dsDatasetName - return a pointer to a dataset name

dsFindEntry - search for a dataset entry by path name

dsIsAcyclic - checks for an acyclic dataset

dsIsDataset - checks for dataset handle

dsIsTable - checks for table handle

dsRefcount - returns the number of links to a dataset or table

dsDatasetEntry

The dsDatasetEntry function returns a handle for an entry in a dataset.

int dsDatasetEntry(DS_DATASET_T **ppEntry, DS_DATASET_T *pDataset, size_t entryNumber);

Parameters

ppEntry
Location to return handle of entry
pDataset
Handle of parent dataset
entryNumber
Index number for entry to be returned

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode. To print error information call dsPerror.

dsIsAcyclic

The dsIsAcyclic function checks for an acyclic dataset. The result is TRUE if the dataset is acyclic.

int dsIsAcyclic(DS_DATASET_T *handle);

Parameters

handle
Handle for the dataset

Return Value

If the dataset is acyclic, the return value is TRUE.

If the dataset is cyclic or is invalid, the return value is FALSE. To get extended error information, call dsErrorCode. To print error information call dsPerror.
