Functions that access table cells

dsCellAddress - return the address of a cell in a table

dsGetCell - copy data from a table cell to a program variable

dsPutCell - copy data from a program variable to a table cell

dsCellAddress

The dsCellAddress function returns the address of a cell in a table.

int dsCellAddress(char **pAddress, DS_DATASET_T *pTable, size_t rowNumber , size_t colNumber);

Parameters

pAddress
Location to return cell address
pTable
Handle for table
rowNumber
Row number
colNumber
Column number

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode. To print error information call dsPerror.

dsGetCell

The dsGetCell function copies data from a cell of a table to a specified address.

int dsGetCell(char *address, DS_DATASET_T *pTable, size_t rowNumber , size_t colNumber);

Parameters

address
Destination address for cell data
pTable
Handle for table
rowNumber
Row number
colNumber
Column number

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode. To print error information call dsPerror.

dsPutCell
