Functions that perform operations on table data

dsEquijoin - perform an equijoin of two tables

dsProjectTable - project columns from one table into a second table

dsTargetTable - create a target table for a join or project operation

dsEquijoin

The dsEquijoin function constructs the equijoin of two tables.

int dsEquijoin(DS_DATASET_T *pJoinTable, DS_DATASET_T *pTableOne, DS_DATASET_T *pTableTwo, char * aliases, char *joinList, char *projectList);.

Parameters

pJoinTable
Handle for join
pTableOne
Handle for first table to be joined
pTableTwo
Handle for second table to be joined
aliases
List of aliases names for tables to be joined
joinList
List that specifies the columns to be joined
projectList
List of columns to be projected into join table

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode. To print error information call dsPerror.

Remarks

dsEquijoin() constructs the equijoin of pTableOne and pTableTwo and returns it in pJoinTable. The columns to be returned in pJoinTable are specified by projectList which has the format '{' select_column_specifier ',' .... '}'. A select_column_specifier is either a single column_name or table_name '.' column_name column_name. A select_column_specifier specifies a column in pJoinTable and a column in either pTableOne or pTableTwo. Columns of pJoinTable that are not specified by a select_column_specifier are matched to a column of pTableOne or pTableTwo. This default selection is done when the column occurs in only one table or occurs in both and the pair are joined. projectList may be NULL if all columns are to be selected by the default rule, such as in a natural join. Table names can be given aliases to distinguish the role they play in a self-join or when two tables have the same name. Aliases are specified by aliases which has the format alias_name1 alias_name2. aliases may be NULL if no aliases are required. Pairs of columns to be joined are specified by joinList which has the format '{' join_pair_specifier ',' ... '}'. A join_pair_specifier is either a single column_name or table_name '.' column_name table_name '.' column_name. column_specifier has the format table_name '.' column_name. A join_pair_specifier specifies a column from pTableOne and a column from pTableTwo that are to be joined. If joinList is NULL, a natural join is performed.

dsProjectTable

The dsProjectTable function projects columns from a source table into a destination table.

int dsProjectTable(DS_DATASET_T *pDst, DS_DATASET_T *pSrc, char *projectList);

Parameters

pDst
Handle for destination table
pSrc
Handle for source table
projectList
List of columns to be projected into join table

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode. To print error information call dsPerror.

Remarks

The columns to be returned in pDst are specified by projectList which has the format '{' select_column_specifier ',' .... '}'. A select_column_specifier is either a single column_name or table_name '.' column_name column_name. A select_column_specifier specifies a column in pDste and a column in pSrc. Columns of pDst that are not specified by a select_column_specifier are matched to a column of pSrc.

See Also

dsTargetTable

dsTargetTable

The dsTargetTable function constructs a target table for a join or project operation.

int dsTargetTable(DS_DATASET_T **ppTable, char *tableName, char *typeName, DS_DATASET_T *parentOne, DS_DATASET_T *parentTwo, char *aliases, char *projectList);

Parameters

ppTable
Location of table handle
tableName
Name of target table
typeName
Name of row type for target table
pParentOne
Handle for first parent table
pParentTwo
Handle for second parent table
aliases
List of aliases names for parent tables
projectList
List of columns for target table

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode. To print error information call dsPerror.

Remarks

The columns of th target table are specified by projectList which has the format '{' select_column_specifier ',' .... '}'. A select_column_specifier is either a single column_name or table_name '.' column_name column_name. A select_column_specifier specifies a column in ppTable and a column in either pParentOne or pParentTwo.

See Also

dsEquijoin, dsProjectTable
