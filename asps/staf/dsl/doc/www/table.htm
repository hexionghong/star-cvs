Functions that return table attributes

dsTableColumnCount - return the number of columns in a table

dsTableDataAddress - return the address of data for a table

dsTableIsType - compares a table row type with a C++ type specifier

dsTableMaxRowCount - return the maximum number of rows for a table

dsTableName - return the name of a table

dsTableRowCount - return the row count for a table

dsTableRowSize - return the size of a row in bytes

dsTableTypeName - return the name for the type of a row

dsTableTypeSpecifier - return a C++ style type specifier for the type of a row

dsTableColumnCount

The dsTableColumnCount function returns the number of columns in a table.

int dsTableColumnCount(size_t *pCount, DS_DATASET_T *pTable);

Parameters

pCount
Location for return of number of columns
pTable
Handle for table

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode. To print error information call dsPerror.

dsTableIsType

The dsTableIsType function compares the row type of a table with a C++ type specifier. The result is TRUE if the types are the same else FALSE.

int dsTableIsType(bool_t *pResult, DS_DATASET_T *pTable, char *specifier);

Parameters

pResult
Location to return result of the compare
pTable
Handle for table
specifier
Type specifier

Return Value
