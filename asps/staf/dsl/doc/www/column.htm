Functions that return column attributes

dsColumnDimCount - return the dimensionally of a column of array type

dsColumnDimensions - return the dimensions of a column of array type

dsColumnElcount - return the number of elements in a column

dsColumnName - return the name of a column in a table

dsColumnSize - return the size of a column in bytes

dsColumnTypeCode - return the type code for elements of a column

dsColumnTypeName - return the type name for elements of a column

dsFindColumn - return column number for a named column

dsColumnDimCount

The dsColumnDimCount function returns the number of dimensions for a column in a table.

int dsColumnDimCount(size_t *pCount, DS_DATASET_T *pTable, size_t colNumber);

Parameters

pCount
Location to return number of dimensions
pTable
Handle for table
colNumber
Column number

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.

See Also

dsColumnDimensions

dsColumnElcount

The dsColumnElcount function returns the number of elements in a column of a table. The number of elements in a column is one for a scalar or the product of the dimensions for an array.

int dsColumnElcount(size_t *pCount, DS_DATASET_T *pTable, size_t colNumber);

Parameters

pCount
Address to return count
pTable
Handle for table
colNumber
Column number

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.

See Also

dsColumnDimCount, dsColumnDimensions

dsColumnSize

The dsColumnSize function returns the size in bytes of a column in a table.

int dsColumnSize(size_t *pSize, DS_DATASET_T *pTable, size_t colNumber);

Parameters

pSize
Array for return of column size
pTable
Handle for table
colNumber
Column number

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.

Remarks

The value of size is for one cell in the column.
