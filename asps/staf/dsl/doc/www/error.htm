Function that return error information

dsErrorCode - returns error codes from dataset library functions

dsPerror - prints error information to stderr

dsErrorCode

The dsErrorCode function returns the error code describing the last error encountered during a call to a dataset library function..

DS_ERROR_CODE_T dsErrorCode(void);

Parameters

None

Return Value

The return value is the callers last-error code value.

dsPerror

The dsPerror function prints a short error message on standard error describing the last error encountered during a call to a dataset library function.

void dsPerror(char *message);

Parameters

message
A message to be printed before the error information.

Return Value

None
