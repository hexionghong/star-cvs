ASU: ANALYSIS SERVICE UTILITIES

General purpose utility object library for STAF.

ASU currently serves two general functions:

  • Utilities for ASPs and PAMs.
    ASU functions and classes are used in other ASPs as "general" utilities.
    (eg. asuAlloc, string)
  • Catchall library.
    The ASU library contains some functions and classes that will eventually be moved into devoted ASPs.
    (eg. EML)
